Configuración del entorno

Antes de iniciar la aplicación, Graphviz debe estar instalado en el sistema y correctamente agregado al PATH. Además se deben instalar algunas librerías externas, necesarias para el funcionamiento del programa:

  1. Instalar Graphviz y agregarlo al PATH.

  2. Abrir una terminal, navegar a una carpeta vacía donde se tenga permisos de escritura y clonar el proyecto desde el repositorio:

    git clone <URL_repositorio.git>

  3. Crear y activar un entorno virtual de Python:

    1. Para crearlo, ejecutar en la carpeta actual:

      python -m venv nombre_del_entorno  # Válido en Linux y Windows

    2. Para activarlo:

      source nombre_del_entorno/bin/activate  # Linux

.nombre_del_entorno/Scripts/activate    # Windows

      Para verificar que está activado, el siguiente comando debería retornar la ruta del entorno creado:

      which python  # Linux

where python  # Windows

  4. Navegar a la carpeta raíz del proyecto (donde se encuentran los archivos dependencias.txt y main.py) e instalar las dependencias:

    pip install -r dependencias.txt  # Linux y Windows

  5. Para iniciar el programa, ejecutar:

    python main.py  # Linux y Windows

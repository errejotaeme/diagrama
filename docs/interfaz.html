cambiar_tema(diagrama_no_vacio: bool, proyecto_activo: tuple[str, str], texto_respaldo: str) → None[fuente]

Alterna entre el tema claro y el oscuro.

Parámetros:

  • diagrama_no_vacio (bool) – Indica si se debe actualizar el artefacto del área de gráfico para que coincida con el color de fondo del grafo.

  • proyecto_activo (tuple[str, str]) – Datos que permiten volver a activar el proyecto que estaba abierto, al reiniciar por cambio en tema.

  • texto_respaldo (str) – Respaldo del contenido del área de texto que se vuelve a cargar cuando se reinicia por cambio de tema.

Módulo interfaz.diseño

class interfaz.diseño.Interfaz(bucle, tema: str, respaldo_pendiente: bool, diagrama_no_vacio: bool, proyecto_activo: tuple[str, str], raiz: Tk, texto_respaldo: str)[fuente]

Bases: object

Clase encargada de construir los componentes básicos de la aplicación.

Parámetros:

  • bucle (Bucle) – Instancia de la clase encargada de crear el bucle de la aplicación y gestionar el cambio de tema.

  • tema (str) – El tema actual de la aplicacion.

  • respaldo_pendiente (bool) – Indica si se debe cargar el respaldo temporal guardado antes del reinicio por cambio de tema.

  • diagrama_no_vacio (bool) – Indica si se debe actualizar el artefacto del área de gráfico para que coincida con el color de fondo del grafo.

  • proyecto_activo (tuple[str, str]) – Datos que permiten volver a activar el proyecto que estaba abierto, al reiniciar por cambio en tema.

  • raiz (Tk) – Ventana principal que controla el bucle de la aplicación.

  • texto_respaldo (str) – Respaldo del contenido del área de texto que se vuelve a cargar cuando se reinicia por cambio de tema.

desplegar() → None[fuente]

Instancia y ordena los elementos en la ventana principal. Da inicio al bucle de la aplicación.

Módulo interfaz.comunes

Módulo que contiene las estructuras de datos usadas para dar estilo a la aplicación y el diagrama. Además almacena las opciones disponibles en los menús y algunos datos requeridos por el Gestor y el área de edición.

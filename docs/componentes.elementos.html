Módulo componentes.elementos.emergente

class componentes.elementos.emergente.NotaEmergente(elemento_asociado, texto: str)[fuente]

Bases: object

Clase encargada de gestionar y generar una nota emergente.

Parámetros:

  • elemento_asociado (Widget) – Artefacto sobre el cual será visible la nota.

  • texto (str) – Contenido de la nota.

mostrar(*args) → None[fuente]

Vuelve visible la nota emergente si está oculta. Args captura los eventos vinculados a los artefactos que llaman a éste método.

ocultar(*args) → None[fuente]

Si está visible, oculta la ventana que contiene a la nota emergente. Args captura los eventos vinculados a los artefactos que llaman a éste método.

Módulo componentes.elementos.opciones

class componentes.elementos.opciones.MenuOpciones(ancestro: Frame, opc_inicial: str, opciones: list[str], atributos: dict[str, str | int], ref_instancia: str, gestor)[fuente]

Bases: object

Clase encargada de construir un OptionMenu y comunicarlo con el Gestor.

Parámetros:

  • ancestro (Frame) – El marco donde se ubicará.

  • opc_inicial (str) – La opción visible cuando se crea el artefacto.

  • opciones (list[str]) – Cada opción del menú desplegable.

  • atributos (dict[str, str | int]) – El estilo del artefacto.

  • ref_instancia (str) – Un nombre que permite al gestor identificar la instancia y enrutar la opción elegida al módulo correspondiente.

  • gestor (Gestor) – La instancia de la clase encargada de gestionar la comunicación entre los módulos y la interacción con las tablas.

_cambio_opc(*args) → None[fuente]

Envía al gestor la opción seleccionada. Args captura el evento de interacción con la instancia del artefacto.

El menú de opciones.

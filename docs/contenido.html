Módulo contenido

class contenido.Extractor[fuente]

Bases: object

Clase encargada de obtener el texto contenido en un pdf o un txt.

extraer_contenido(ruta: str) → str[fuente]

Obtiene el texto completo del archivo que recibe como argumento. Si no lo obtiene retorna -1.

Parámetros:

ruta (str) – Ubicación del archivo.

Devuelve:

El texto contenido en el archivo o -1.

Tipo del valor devuelto:

str

ordenar_contenido(contenido: str) → str[fuente]

Convierte las líneas extraídas en oraciones.

Parámetros:

contenido – El texto del PDF.

Devuelve:

Las líneas del PDF concatenadas en oraciones.

Tipo del valor devuelto:

str
